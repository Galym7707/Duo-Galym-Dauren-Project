## Numbers Guidance

If you need one numbers block on a slide:

- use `tCO2e`
- use `time to triage`
- use `verification ownership`

Do not invent:

- exact KZT savings
- exact leak quantities without source support
- precise ROI percentages

Use formulas instead:

`Value = faster triage + lower reporting friction + reduced exposure from delayed action`
